# Victory Charts

Did you know that making charts with **Victory** was *this easy*?

::::columns

:::column{ grow=true }
```tsx
import { VictoryPie } from "victory-bar";

const MyComponent = () => (
  <VictoryPie
    data={[ { x: "Cats", y: 40 }, /* ... */ ]}
    colorScale={["gold", "orange", "navy" ]}
    innerRadius={60}
    style={{ labels: { fontSize: 25 } }}
  />
);
```
:::
